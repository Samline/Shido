Usage

Quick start guide for use and configuring colors palette.

Using color variables

The color palette is generated from internal mixins, if you want to use the color variables, i recommend having Shido within your build process. You can see the full list of colors here, all available colors are shown there.

This is an example of how to use the default variables, just choose a color and you're done.


.container {
  background-color: @c-grey-200;
  color: @c-blue-600;
}

You can declare your own color variables using existing ones.


// Grey
@c-grey-light: @c-grey-300;
@c-grey:       @c-grey-500;
@c-grey-dark:  @c-grey-700;

      

.container {
  background-color: @c-grey-light;
  color: @c-grey-dark;
}

Utility color classes

If you want to have the color palette directly in your html, Shido has two prefixes that you can use: bg- and c-, all you have to do is choose a color.


<div class="bg-teal-500">
  <p class="c-cyan-50">Hello world!</p>
</div>

Hello world!

Custom classes

You can generate your own classes with the color mixin.


// Mixins
.mix-(bg/color/bg-color)-class(color, name);

Just choose a color and the class name.


// Example
.mix-bg-class(#9c27b0, my-custom-background);
.mix-color-class(#ffecb3, my-custom-color);

      

<div class="bg-my-custom-background">
  <p class="c-my-custom-color">Hello world!</p>
</div>

Hello world!

You can generate both classes at the same time using the following mixin.


// Example
.mix-bg-color-class(#ffb300, my-custom-bg-and-color);

      

<div class="bg-my-custom-bg-and-color">
  <p>Hello world!</p>
</div>

Hello world!


<div>
  <p class="c-my-custom-bg-and-color">Hello world!</p>
</div>

Hello world!
