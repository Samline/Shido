How it works

All details about the inside of Shido will be documented in this section.

Details

  • Developed with Less
  • Number of columns: 12
  • Column padding: 10px
  • Offsets: Yes
  • Column ordering: No
  • Default font size: 16px
  • Safe area: 1280px

Breakpoints

The breakpoints apply for devices with screen widths greater than or equal to the defined breakpoints, invalidating the breakpoints for smaller devices.

For example using columns, any class .m:scol-* will work on medium and large devices if a class .l:scol-* is not present.

x

≥ 0px

s

≥ 600px

m

≥ 768px

l

≥ 1024px

xl

≥ 1280px

It is important to note that the breakpoint x is applied by default in some cases such as columns and fonts.

Columns

If you are using a class .s:scol-4 and there is no class .x:scol-* automatically the breakpont by default it will be .x:scol-12, if this is what you want then it is not necessary to declare the class .x:scol-12


<div classs="line">
  <div class="s:scol-4">Content</div>
  <div class="s:scol-4">Content</div>
  <div class="s:scol-4">Content</div>
</div>

If you are using a class .s:fs-24 and there is no class .x:fs-* automatically the breakpont by default it will be .x:fs-16, if this is what you want then it is not necessary to declare the class .x:fs-16


<p class="s:fs-24">Content</p>

Variables

With the exception of breakpoints, all variables begin with the prefix s- 

// Breakpoints
@x-screen-min:  0px;
@x-screen-max:  599px;
@s-screen-min:  600px;
@s-screen-max:  767px;
@m-screen-min:  768px;
@m-screen-max:  1023px;
@l-screen-min:  1024px;
@l-screen-max:  1279px;
@xl-screen-min: 1280px;

With breakpoint variables you can set media queries to give specific behaviors.


@media (max-width: @m-screen-max) { ... }
@media (min-width: @l-screen-min) { ... }

      

// Fonts
@s-increment-font: 2;
@s-min-font-size:  10;
@s-max-font-size:  60;

// Grid
@s-scontainer:  1280px;
@s-columns:     12;
@s-grid:        100 / @s-columns;
@s-gutter:      10px;
@s-clear-space: @s-body-font-size;

// Marging & Padding
@s-max-margin:  8; // x10
@s-max-padding: 8; // x10

// Layout
@s-body-background-color: #fff;
@s-body-font-color:       #000;
@s-headers-font-color:    #000;
@s-body-font-family:      -apple-system, BlinkMacSystemFont, "Segoe UI", "Source Sans Pro", Oxygen, sans-serif;
@s-headers-font-family:   -apple-system, BlinkMacSystemFont, "Segoe UI", "Source Sans Pro", Oxygen, sans-serif;
@s-body-font-size:        16px;
@s-main-link-color:       #2196F3;
@s-main-link-color-hover: #1565C0;

// Font weight
@s-font-weight-thin:        200;
@s-font-weight-light:       300;
@s-font-weight-regular:     400;
@s-font-weight-medium:      500;
@s-font-weight-semi-bold:   600;
@s-font-weight-bold:        700;
