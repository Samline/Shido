Fonts

Shido has two ways to set the size of a font: responsive and static.

Choose the font size in pixels and the resulting size will be its equivalent in rem according to the default font size @s-body-font-size

Responsive fonts

Responsive fonts are utilitarian classes to be used directly in your html. By default the available font sizes are from 10 to 60, but you can modify them according to your needs.

Only even numbers are available. 

<p class="x:fs-18 s:fs-22 m:fs-24 l:fs-30 xl:fs-40">Hello world!</p>

Hello world!

Static fonts

Static fonts are generated from mixins so they must be compiled.

Any font size is accepted. 

.example-hello-world {
  color: @c-teal-500;
  .mix-fs(36); /* Return font-size: 2.25rem */
}

      

<p class="example-hello-world">Hello world!</p>

Hello world!
