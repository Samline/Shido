Vertical alignment

  • *:baseline
  • *:bottom
  • *:middle
  • *:top

Vertical alignment is intended to be used in columns, but can also be used in any inline-block type element.

*:scol- and *:scol-fit- by default are vertically centered. 

<div class="line">
  <div class="x:scol-4">Content</div>
  <div class="x:scol-4">Content</div>
  <div class="x:scol-4">Content</div>
</div>









You can align to the top.


<div class="line">
  <div class="x:scol-4 x:top">Content</div>
  <div class="x:scol-4 x:top">Content</div>
  <div class="x:scol-4 x:top">Content</div>
</div>









You can align to the bottom.


<div class="line">
  <div class="x:scol-4 x:bottom">Content</div>
  <div class="x:scol-4 x:bottom">Content</div>
  <div class="x:scol-4 x:bottom">Content</div>
</div>









Or you can combine them too.


<div class="line">
  <div class="x:scol-4 x:bottom">Content</div>
  <div class="x:scol-4 x:bottom">Content</div>
  <div class="x:scol-4 x:top">Content</div>
</div>
